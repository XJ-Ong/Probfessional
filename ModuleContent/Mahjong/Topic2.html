Understanding Sampling Without Replacement

Mahjong demonstrates a crucial probability concept: sampling without replacement. Unlike dice where each roll is independent, Mahjong tiles are removed from the deck once drawn, affecting all subsequent probabilities.

How It Works

  • Initial Probability: P(bamboo tile) = 36/144 = 1/4 at start
  • After Drawing One: P(bamboo) = 35/143 if first was bamboo, or 36/143 if first was not
  • Each draw changes the remaining pool and probabilities

Consecutive Draws

  • P(drawing two bamboo tiles in a row) = (36/144) × (35/143) ≈ 0.0616
  • P(drawing dragon then flower) = (12/144) × (8/143) ≈ 0.0047
  • Order matters in these calculations!

Strategic Implications

  • As tiles are drawn, probabilities update continuously
  • Players must recalculate odds throughout the game
  • Knowing which tiles have been played affects future draws

Application: This concept applies to many card games and real-world sampling scenarios!
