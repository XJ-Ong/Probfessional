Understanding Independence

Each spin on a slot machine is completely independent - previous spins do NOT affect future outcomes.

Key Concept: The "gambler''s fallacy" is wrong - if cherries appeared once, P(cherries next spin) is still the same!

Basic Probabilities:

  • If P(cherry) = 1/10 for each reel
  • P(three cherries) = (1/10) × (1/10) × (1/10) = 1/1000
  • P(not cherry) = 1 - 1/10 = 9/10

Multiplication Rule: For independent events: P(A and B) = P(A) × P(B)

Practice:

  • P(three cherries) = (1/10)³ = 1/1000
  • P(not cherry) = 9/10
  • P(three of same symbol, 5 symbols) = (1/5)³ = 1/125
