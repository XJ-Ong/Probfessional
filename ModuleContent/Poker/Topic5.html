Drawing Multiple Cards

When drawing cards without replacement, the deck size decreases with each draw.

Probability of a Pair:

  • First card: any card (52/52 = 1)
  • Second card matching: 3 remaining cards of same rank out of 51 cards = 3/51
  • P(pair) = 3/51 ≈ 1/17

Same Suit Probability:

  • First card spade: 13/52 = 1/4
  • Second card spade: 12/51
  • P(both spades) = (13/52) × (12/51) = (1/4) × (12/51) = 12/204 = 1/17

Sequential Drawing:

  • Order matters: P(King then Queen) = (4/52) × (4/51)
  • For order: P = (4/52) × (4/51) × 2 = 2/1326 (accounting for both orders)

Practice:

  • P(pair in two cards) = 3/51 ≈ 1/17
  • P(both spades) = 1/17
  • P(King then Queen, order matters) = 2/1326
