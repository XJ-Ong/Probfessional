Strategic Probability in UNO

Color Matching:

  • With 7 cards in hand, probability of having a matching color depends on deck composition
  • If top card is red, P(you have red card) ≈ 1/4

Special Cards:

  • P(top card is not a number card): 32 special cards / 108 = 32/108
  • P(drawing skip card): 8/108 = 2/27

Game State Impact:

  • As cards are played, probabilities change
  • If you have 3 yellow cards, P(opponent has no yellows) depends on remaining deck
  • Later in game, probabilities shift significantly

Practice:

  • P(opponent can play if top card is red) ≈ 1/4
  • P(drawing skip card) = 8/108 = 2/27
  • P(all number cards gone, then wild) = 8/32 = 1/4
